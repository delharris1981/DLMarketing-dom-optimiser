# DLMarketing DOM Optimiser

**High-performance DOM cleaner for Elementor (Legacy & V4).**

## Overview
This plugin reduces DOM depth and removes "bloat" markup generated by page builders like Elementor. It operates on the frontend output buffer, meaning it **never modifies your actual database content**. It creates a leaner, faster-loading page on the fly.

## Features
*   **Wrapper Pruning:** Intelligently collapses redundant `.elementor-column-wrap`, `.elementor-widget-wrap`, and `.elementor-widget-container` divs when they serve no semantic or visual purpose.
*   **Ghost Node Detection:** Removes empty Elementor widgets/containers often left behind by conditional logic or empty Dynamic Tags.
*   **Comment Cleanup:** Strips HTML comments to reduce page size.
*   **Safety Rails:**
    *   **Visual Protection:** Does not remove opaque elements (elements with backgrounds/borders - *Basic detection in Alpha*).
    *   **User Class Integrity:** Any element with a custom class (not starting with `elementor-`) is locked and preserved.
    *   **Interaction Safe:** Preserves all elements with `id`, `onclick`, `tabindex`, or `role` attributes to ensure JS and Accessibility remain intact.

## Installation
1.  Upload the plugin folder to `wp-content/plugins/`.
2.  Activate via WordPress Admin.
3.  The plugin currently runs in "Auto" mode (Tier 1 & 2 safe optimizations).

## Compatibility
*   **WordPress**: 6.0+
*   **PHP**: 7.4+
*   **Elementor**: 3.0+ (Fully compatible with V4 Atomic Architecture)
